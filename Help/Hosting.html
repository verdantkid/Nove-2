Hosting Servers

to host servers on a VPS or your own pc

visit here

you need to have docker installed for this

you can use WSL if you are using Windows, or use the terminal if you're using linux

also please install docker on a disk drive that has alot of space (around 10 gigs of free space), or else it will get errors while building

if you're running the server and it doesn't show anything up after that, please type: apt-get install docker-buildx-plugin

and then rebuild the docker image
